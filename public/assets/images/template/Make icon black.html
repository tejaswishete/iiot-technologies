To make the icons in black color, you need to ensure a few things:

✅ 1. Use SVG icons or icons that support color changes

If your panel.lightIcon and panel.darkIcon are SVGs, you can change their color using fill: black.

However, if they are PNG or JPG, then they are static images, and their color can't be changed with CSS. In that case, you need to either:

  • Use a black version of the icon image, or

  • Convert them to SVG format if you want dynamic color control.

🛠️ Option 1: If using SVG (and stored as React components or inline SVG)

Instead of <Image>, do:

jsx
<svg
  className="w-64px lg:w-80px"
  fill="black"
  xmlns="http://www.w3.org/2000/svg"
  viewBox="0 0 24 24"
>
  {/* SVG path here */}
</svg>

🛠️ Option 2: If using Next.js <Image /> and the icons are static black icons

Just replace your current image paths with black icon versions:

jsx
<Image
  className="w-64px lg:w-80px"
  src={panel.blackIcon} // You need to add this to your panelData
  width={100}
  height={100}
  alt={panel.altText}
/>

And make sure your panelData has blackIcon key with path to black version.

🛠️ Option 3: Apply CSS filter (if you're forced to use colored PNGs and want to make them appear black)

jsx
<Image
  className="w-64px lg:w-80px filter grayscale brightness-0"
  src={panel.lightIcon}
  width={100}
  height={100}
  alt={panel.altText}
/>

This will desaturate and darken the image to look black. Not perfect for all icons.
